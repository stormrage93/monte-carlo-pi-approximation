# monte-carlo-pi-approximation
An investigation into the Monte Carlo simulation as applied to pi approximation. "Explore the full analysis and visualizations in the pi_approximation.ipynb file."

## Overview
The Monte Carlo simulation can be used to approximate $\pi$. This is accomplished by inscribing a circle within a unit square and taking ratios of sampling points that fall within the circle verus the total number of points sampled in the square. However, given the introductory nature, the investigation usually ends here. This project is an exploration of the effects on the approximation when the number of sample points is altered as well as extensions of the simulation into higher dimensions.
